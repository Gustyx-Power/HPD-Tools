# Services

This folder contains the core business logic services for the FiveM Police Broadcast Overlay application.

## Planned Services

- **ConfigurationManager**: Handles loading, saving, and managing application configuration
- **ProcessMonitor**: Monitors for FiveM process start/stop events
- **KeybindManager**: Manages global keyboard hooks and keybind registration
- **MessageSender**: Formats and sends messages to FiveM chat via keyboard simulation
- **OverlayManager**: Manages overlay window lifecycle and visibility

Services will be implemented in subsequent tasks.
